使用教程

首先解压到 任意位置

注意:这个Mod Api 【20180429 为《森林》游戏版本的专用版（还有其他版本的游戏和工具版本号）,


然后把 MOD 放进 TheForest 文件里

运行 modapi.exe（你可以为该.exe创建一个快捷方式，放在你想放的地方）

勾选对应的 MOD ，为地图MOD绑定快捷键，点击右下角的 启动游戏（注意不要在Steam启动游戏！）

因为该MOD是汉化版，所以我在这里就不再赘述游戏内的使用教程，还请自行摸索
